# Daily-coding-growth
记录每日编程学习内容、代码练习、成长复盘.......

📝
NK: 牛客

🎯 *****核心目标****
夯实基础：从语法到算法，系统性补全计算机基础知识
刻意练习：每日保持代码手感，用输出倒逼输入
沉淀复盘：将零散知识点整理成可复用的笔记与模板
长期成长：记录从编程新手到合格开发者的完整进阶路径

📂 ****结构说明****

plaintext
Daily-coding-growth/
├── 01-笔记汇总/       # 学习笔记、语法总结、易错点整理
├── 02-数据结构/        # 数组、链表、栈、队列、树等实现与解析

├── 04-成长复盘/        # 阶段总结、学习规划与反思

✨ ****风格与规范****
代码：遵循 C/C++ 工程规范，变量命名清晰，关键逻辑加注释
笔记：使用 Markdown 格式，结构清晰，重点突出
提交：小步提交，信息明确，便于回溯与复盘
