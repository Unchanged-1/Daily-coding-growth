# 我给你推荐几款在 VS Code 里记 Markdown 笔记的「神器级插件」，按功能分类，方便你按需安装：
## 📝核心编辑增强
###  Markdown All in One
功能：一键格式化表格、自动生成目录、快捷键支持（加粗 / 斜体 / 列表）、预览同步滚动。
适合：日常记笔记，提升编辑效率。
### Markdown Extended
功能：支持更多语法（如脚注、定义列表、任务列表），增强原生 Markdown 能力。
适合：需要复杂排版的笔记。
## 🎨 预览与美化
### Markdown Preview Enhanced
功能：强大的预览功能，支持导出 PDF/HTML、数学公式渲染、自定义 CSS 样式。
适合：需要导出或美化笔记的场景。
### Markdown Theme Kit
功能：提供多种 Markdown 预览主题，让笔记更美观。
适合：追求视觉体验的用户。
## 🧩 结构化与管理
### Foam
功能：类似 Obsidian 的双链笔记、图谱视图、反向链接，适合构建知识网络。
适合：构建个人知识库、学习笔记。
### Dendron
功能：基于层级的笔记管理，支持快速查找和链接，适合大量笔记的组织。
适合：程序员、学生等需要管理大量学习笔记的用户。
## 🛠️ 辅助工具
### Paste Image
功能：直接从剪贴板粘贴图片到 Markdown，自动保存并生成链接。
适合：需要插入大量截图的笔记
### Markdownlint
功能：检查 Markdown 格式，给出规范建议（你之前遇到的警告就是它）。
适合：追求格式规范或分享笔记的用户。
💡 我的推荐组合
日常记笔记：Markdown All in One + Paste Image
构建知识库：Foam + Markdown Preview Enhanced
分享 / 开源文档：Markdownlint + Markdown All in One
